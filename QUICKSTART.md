# Быстрый старт

## 🚀 Запуск расширения за 2 минуты

### Шаг 1: Подготовка
- Убедитесь, что у вас установлен Chrome или Edge (версия 88+)
- Откройте папку `steam-price-rub` в любом текстовом редакторе

### Шаг 2: Загрузка расширения
1. Откройте Chrome и введите в адресной строке: **`chrome://extensions/`**
2. В правом верхнем углу нажмите переключатель **"Режим разработчика"**
   - Должна появиться кнопка "Загрузить распакованное расширение"
3. Нажмите эту кнопку и выберите папку `steam-price-rub`
4. Готово! Расширение установлено ✓

### Шаг 3: Тестирование
1. Откройте любую страницу Steam (например: https://steampowered.com/app/570/)
2. Посмотрите на цену игры - рядом должна появиться цена в рублях синим цветом
3. Нажмите на иконку расширения (💰) в правом верхнем углу Chrome для настроек

## 🔧 Основные функции

### Автоматическая конвертация
- Расширение автоматически находит все цены на странице Steam
- Рядом с каждой ценой в USD показывает конвертированную цену в RUB
- Работает на всех страницах Steam (каталог, карточки игр, бандлы, скидки)

### Попап-меню (нажмите на иконку 💰)
```
┌─ Курс USD/RUB: 100.50
├─ Обновлено: 5м назад
├─ [Обновить курс сейчас] - ручное обновление
├─ [Параметры] - настройки
└─ Информация о расширении
```

### Автоматические обновления
- Курс обновляется автоматически каждые 30 минут
- Используется бесплатный API ExchangeRate-API
- Последний известный курс сохраняется локально

## ❓ FAQ

### Почему цены не обновляются?
**Решение:**
- Перезагрузите страницу Steam (Ctrl+F5)
- Проверьте, что расширение активно в chrome://extensions/
- Попробуйте обновить курс вручную через попап

### Как отключить расширение?
1. Откройте chrome://extensions/
2. Найдите "Steam Price RUB"
3. Выключите переключатель или нажмите "Удалить"

### Курс неправильный
- Нажмите "Обновить курс сейчас" в попап-меню
- Проверьте интернет-соединение
- Убедитесь, что API https://api.exchangerate-api.com/ доступен

### Работает ли на других сайтах?
В данный момент расширение настроено только для Steam. Если вы хотите расширить функциональность на другие сайты, отредактируйте `manifest.json`:

```json
"host_permissions": [
  "https://steampowered.com/*",
  "https://*.steampowered.com/*",
  "https://другойсайт.com/*"  // Добавьте сюда
]
```

## 💡 Советы и хитрости

### Совет 1: Обновите курс перед крупной покупкой
Если вы покупаете дорогую игру, нажмите "Обновить курс сейчас" для точного расчёта.

### Совет 2: Часовой пояс
Время обновления показывается в локальном часовом поясе вашего компьютера.

### Совет 3: Отключите для конкретной вкладки
Если расширение мешает работе Steam (редко), откройте меню расширения и выберите "Управление расширениями" → "На этом сайте"
