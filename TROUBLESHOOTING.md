# TROUBLESHOOTING - Решение проблем

## ❌ Расширение не работает

### Проверка 1: Убедитесь, что расширение установлено

1. Откройте `chrome://extensions/`
2. Убедитесь, что **Steam Price RUB** есть в списке
3. Если расширения нет - установите его заново (см. README.md)

### Проверка 2: Убедитесь, что расширение включено

1. Откройте `chrome://extensions/`
2. Найдите **Steam Price RUB**
3. Убедитесь, что переключатель рядом с ним **включен** (синий)
4. Если выключено - нажмите на переключатель

### Проверка 3: Перезагрузите страницу

- Просто нажмите **F5** на странице Steam
- Или нажмите **Ctrl+Shift+R** (очистить кэш и перезагрузить)

### Проверка 4: Вы на сайте Steam?

Расширение работает только на:
- `https://steampowered.com`
- `https://store.steampowered.com`
- `https://community.steampowered.com`

Проверьте адресную строку - там должно быть `steampowered.com`

## ❌ Цены не конвертируются

### Причина 1: Расширение не запустилось

1. Откройте консоль разработчика (нажмите **F12**)
2. Перейдите на вкладку **Console**
3. Посмотрите, есть ли там ошибки (красные сообщения)
4. Если есть - скопируйте сообщение об ошибке

### Причина 2: Курс не загружается

Если в попап-меню вместо курса показано "-":

1. Нажмите кнопку "Обновить курс сейчас"
2. Подождите 2-3 секунды
3. Если курс обновился - проблема в API соединении
4. Проверьте интернет-соединение

### Причина 3: На странице нет стандартных цен

Некоторые страницы Steam могут отображать цены нестандартным образом. 
Попробуйте:
- Перейти на другую страницу Steam (например, главную страницу магазина)
- Если там работает - значит на конкретной странице нестандартный формат цен

## ⚠️ API недоступен (ошибка сети)

### Признаки:
- В попап показано: "Ошибка: Network error"
- Цены не конвертируются вообще

### Решение:

**Вариант 1: Проверьте интернет**
```
1. Откройте обычный браузер (без расширения)
2. Попробуйте открыть https://api.exchangerate-api.com/v4/latest/USD
3. Если сайт не откроется - проблема с вашим интернетом
```

**Вариант 2: API может быть недоступен**
- Сервис ExchangeRate-API может быть временно не доступен
- Попробуйте позже
- Расширение будет использовать последний известный курс

**Вариант 3: Блокировка расширения VPN/антивирусом**
- Отключите VPN временно
- Проверьте настройки антивируса/брандмауэра

## 🔴 Расширение вызывает ошибки на Steam

### Признаки:
- Steam работает медленно
- Ошибки в консоли (F12)
- Элементы страницы не работают

### Решение:

**Шаг 1: Отключите расширение**
1. Откройте `chrome://extensions/`
2. Выключите **Steam Price RUB**
3. Перезагрузите страницу Steam (F5)

**Шаг 2: Проверьте версию Chrome**
- Убедитесь, что у вас Chrome 88 или новее
- Откройте `chrome://settings/help` чтобы проверить версию

**Шаг 3: Переустановите расширение**
1. Откройте `chrome://extensions/`
2. Нажмите "Удалить" рядом с расширением
3. Установите его заново

---

## 📊 Курс неправильный

### Признаки:
- Курс сильно отличается от текущего (например, показывает 50 RUB вместо 100+)
- Цены конвертируются неправильно

### Решение:

**Шаг 1: Обновите курс вручную**
1. Нажмите на иконку расширения (💰) в Chrome
2. Нажмите кнопку "Обновить курс сейчас"
3. Подождите обновления

**Шаг 2: Проверьте время последнего обновления**
- В попап должно быть написано "Обновлено: ... назад"
- Если там "никогда" или очень давно - требуется обновление

**Шаг 3: Очистите кэш расширения**
1. Откройте консоль (F12)
2. Перейдите на вкладку Application → Local Storage
3. Найдите запись с `steam-price-rub` или `exchangeRate`
4. Удалите её
5. Перезагрузите страницу

## 🔐 Расширение просит доступ к моим данным

### Объяснение:

Расширение запрашивает доступ к:
- **Чтению содержимого страниц** - для поиска цен
- **API ExchangeRate** - для получения курса обмена

Это **НОРМАЛЬНО** и необходимо для работы.

Расширение **НЕ** имеет доступа к:
- Вашей истории Steam
- Пароли и логины
- Личные сообщения
- Покупки и платежные данные

## 🐢 Расширение замедляет браузер

### Причины:
- Много цен на странице
- Медленное интернет-соединение
- Слабый компьютер

### Решение:

**Вариант 1: Отключите на конкретном сайте**
1. Нажмите на иконку расширения (💰)
2. Нажмите "Управление расширениями"
3. Найдите Steam Price RUB
4. Нажмите на "Подробнее"
5. Выберите "На этом сайте" вместо "На всех сайтах"

**Вариант 2: Уменьшите область обработки**
Отредактируйте `content.js` и уменьшите количество селекторов
