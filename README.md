# Steam RUB

Минимальное Chrome-расширение для `store.steampowered.com`.

Расширение сканирует текст на открытой странице, автоматически определяет валюту цены и добавляет рядом цену в рублях.

Пример:

```text
100₴ -> 100₴ (220 ₽)
$10 -> $10 (1 000 ₽)
```

## Возможности

- минимальная структура проекта
- работа на `store.steampowered.com`
- сканирование текста на текущей странице
- автоматическое определение нескольких валют
- конвертация цен в `₽`

## Структура проекта

```text
steam-price-rub/
├── manifest.json
├── content.js
└── README.md
```

## Установка

1. Откройте `chrome://extensions`
2. Включите режим разработчика
3. Нажмите `Загрузить распакованное расширение`
4. Выберите папку проекта

## Как это работает

- расширение ищет цены в разных форматах, например `33₴`, `$10`, `€5`
- курсы валют загружаются через бесплатный API
- исходная цена сохраняется
- рядом дописывается сумма в рублях

## Ограничения

- это простая MVP-версия
- при недоступности API используется последнее сохранённое значение
- сейчас поддерживаются `₴`, `$`, `€`, `£`, `₸`, `₺`, `PLN`, `zł`, `¥`
