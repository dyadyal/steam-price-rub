# Steam RUB

Минималистичное Chrome-расширение для `store.steampowered.com`, которое автоматически находит цены на странице, определяет валюту и показывает рядом сумму в рублях.

## Пример

```text
100₴ -> 100₴ (220 ₽)
$10  -> $10 (1 000 ₽)
€5   -> €5 (550 ₽)
```

## Что умеет

- работает прямо на страницах Steam
- сканирует текст на открытой странице
- автоматически определяет валюту цены
- добавляет рядом сумму в рублях
- загружает курсы через бесплатный API
- использует сохранённые курсы, если API временно недоступен

## Поддерживаемые валюты

- `₴`
- `$`
- `€`
- `£`
- `₸`
- `₺`
- `PLN`
- `zł`
- `¥`

## Установка

1. Откройте `chrome://extensions`
2. Включите режим разработчика
3. Нажмите `Загрузить распакованное расширение`
4. Выберите папку проекта

## Как это работает

1. Расширение запускается на `store.steampowered.com`
2. Загружает актуальные курсы валют через бесплатный API
3. Сканирует текст на странице
4. Находит цены в поддерживаемых валютах
5. Добавляет рядом сумму в рублях

## Ограничения

- это минимальная MVP-версия
- точность зависит от формата цены на странице
- некоторые нестандартные варианты записи цены могут потребовать отдельной обработки
