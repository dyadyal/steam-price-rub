# Steam Price RUB - Расширение Chrome

Расширение, которое автоматически конвертирует все цены на сайте Steam в русские рубли.

## Функции

- ✅ Автоматическая конвертация цен из USD в RUB на всех страницах Steam
- ✅ Поддержка цен с скидками и распродажах
- ✅ Конвертирует цены в каталоге, внутриигровых магазинах и бандлах
- ✅ Автоматическое обновление курса каждые 30 минут
- ✅ Использует бесплатный API (ExchangeRate-API)
- ✅ Работает на основе Manifest V3 (современный стандарт Chrome)

## Установка

### Метод 1: Разработчикский режим Chrome (рекомендуется для разработки)

1. Откройте Chrome и перейдите на `chrome://extensions/`
2. Включите **"Режим разработчика"** (переключатель в правом верхнем углу)
3. Нажмите **"Загрузить распакованное расширение"**
4. Выберите папку с этим проектом (`steam-price-rub`)
5. Готово! Расширение загружено и готово к использованию

### Метод 2: Установка из файла .crx (для конечных пользователей)

1. Откройте `chrome://extensions/`
2. Включите **"Режим разработчика"**
3. Перетащите файл `.crx` на страницу расширений
4. Подтвердите установку

## Как использовать

1. Откройте любую страницу Steam (магазин игр, карточку игры, профиль и т.д.)
2. Расширение автоматически найдёт все цены
3. Рядом с каждой ценой в USD появится конвертированная цена в RUB (синим цветом)
4. Нажмите на иконку расширения в Chrome, чтобы:
   - Посмотреть текущий курс обмена
   - Обновить курс вручную
   - Настроить параметры

## Как это работает

1. **content.js** - запускается на всех страницах Steam и:
   - Находит все элементы с ценами
   - Извлекает числовое значение цены
   - Получает текущий курс USD→RUB
   - Добавляет рядом конвертированную цену

2. **background.js** - фоновый сервис, который:
   - Обновляет курс каждые 30 минут
   - Хранит курс в локальном хранилище
   - Обрабатывает запросы от контент-скрипта

3. **popup.html/popup.js** - интерфейс расширения:
   - Показывает текущий курс
   - Позволяет вручную обновить курс
   - Предоставляет основные параметры

## Технические детали

- **API**: ExchangeRate-API (https://exchangerate-api.com/)
- **Версия Manifest**: 3 (Manifest V3)
- **Поддерживаемые браузеры**: Chrome 88+, Edge 88+, Opera 74+

## Кэширование

Расширение кэширует курс обмена на 1 час:
- При первом открытии Steam загружается актуальный курс
- В течение 1 часа используется кэшированный курс
- Автоматическое обновление происходит каждые 30 минут
- Вы можете вручную обновить курс через попап

## Возможные проблемы

### Цены не конвертируются
- Убедитесь, что расширение включено (иконка должна быть видна)
- Перезагрузите страницу Steam (F5)
- Проверьте, что вы находитесь на сайте steampowered.com

### API недоступен
- Если сервис экстернального API недоступен, используется последний известный курс
- Попробуйте обновить курс вручную через попап
- Проверьте ваше интернет-соединение
