# Steam RUB

`Steam RUB` — минималистичное Chrome-расширение для `store.steampowered.com`, которое находит цены на странице и показывает рядом ориентировочную сумму в рублях.

## Что делает расширение

- работает прямо на страницах Steam Store
- распознаёт цены в нескольких валютах
- добавляет рядом сумму в рублях
- загружает актуальные курсы через `open.er-api.com`
- использует сохранённые курсы, если API временно недоступен

## Поддерживаемые валюты

- `₴`
- `$`
- `€`
- `£`
- `₸`
- `₺`
- `PLN`
- `zł`
- `¥`

## Поддерживаемые форматы цен

Расширение умеет обрабатывать не только простые значения вроде `$59.99`, но и цены с разделителями тысяч:

```text
100₴ -> 100₴ (220 ₽)
2 299,00₴ -> 2 299,00₴ (5 058 ₽)
$59.99 -> $59.99 (5 999 ₽)
1 234 PLN -> 1 234 PLN (32 084 ₽)
```

Также учитываются обычные пробелы, неразрывные пробелы и узкие неразрывные пробелы внутри суммы.

## Что уже исправлено

- корректный разбор цен вида `2 299,00₴`, которые раньше могли читаться как `299,00₴`
- защита от повторной обработки текста, из-за которой одна и та же цена могла несколько раз получить приписку в рублях
- сохранение работы на динамически обновляемых блоках страницы

## Установка

1. Откройте `chrome://extensions`
2. Включите режим разработчика
3. Нажмите `Загрузить распакованное расширение`
4. Выберите папку проекта

## Как это работает

1. Расширение запускается на страницах `https://store.steampowered.com/*`
2. Загружает курсы валют и кеширует их в `chrome.storage.local`
3. Сканирует текстовые узлы страницы
4. Находит цены в поддерживаемых форматах
5. Добавляет рядом эквивалент в рублях
6. Отслеживает динамические изменения страницы через `MutationObserver`

## Ограничения

- это лёгкая MVP-версия без интерфейса настроек
- точность зависит от доступности API и формата записи цен на странице
- если страница уже содержит сложные составные блоки с несколькими валютами в одном тексте, могут потребоваться дополнительные правила обработки
