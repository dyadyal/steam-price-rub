Steam RUB

Минимальное Chrome-расширение для store.steampowered.com, которое автоматически находит цены в разных валютах и показывает рядом сумму в рублях.

100₴ → 100₴ (220 ₽)
$10 → $10 (1 000 ₽)
Открыть GitHub Читать README

Что умеет

  • Сканирует текст на открытой странице Steam
  • Автоматически определяет валюту цены
  • Добавляет рядом сумму в рублях
  • Использует бесплатный API для курсов валют

Поддержка валют

  • ₴
  • $
  • €
  • £
  • ₸
  • ₺
  • PLN / zł
  • ¥

Как установить

  1. Откройте chrome://extensions
  2. Включите режим разработчика
  3. Нажмите «Загрузить распакованное расширение»
  4. Выберите папку проекта

Проект находится в стадии MVP и будет постепенно улучшаться.
