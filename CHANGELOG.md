# CHANGELOG

## [1.0.0] - 2026-04-19

### ✨ Начальный релиз

#### Добавлено
- Автоматическая конвертация всех цен USD в RUB на сайте Steam
- Поддержка различных типов цен:
  - Основные цены игр
  - Цены с учетом скидок
  - Цены в специальных предложениях
  - Цены в бундлах
  - Цены во встроенных игровых магазинах
- Умное кэширование курса обмена (1 час)
- Автоматическое обновление курса каждые 30 минут
- Попап-меню с показом текущего курса и времени последнего обновления
- Возможность ручного обновления курса
- Использование бесплатного API (ExchangeRate-API)
- Поддержка Manifest V3 (современный стандарт)
- Темная тема в интерфейсе
- Локализация на русском языке

#### Технические детали
- **API**: ExchangeRate-API
- **Архитектура**: Manifest V3
- **Браузеры**: Chrome 88+, Edge 88+, Opera 74+
- **Размер**: < 50 KB (без зависимостей)

## Версионирование

Мы используем [Semantic Versioning](https://semver.org/):
- **MAJOR** версия - полная переделка или несовместимые изменения
- **MINOR** версия - добавление новых функций
- **PATCH** версия - исправление багов

Формат: `MAJOR.MINOR.PATCH`

Пример: `1.0.0` = основная версия 1, без новых функций, без исправлений
